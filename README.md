## Getting Started

To get a local copy up and running, follow these simple steps.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Bhbored/gpt_clone
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
